# Word Counter

## How to run the application

1. Make sure `GOBIN` is properly set up in your computer

2. Clone the application with `git clone https://github.com/bmulobi/word-counter.git`

3. Navigate to the directory you have cloned i.e word-counter

4. Run `go install wordcounter`

5. There are several ways to run the app:

> 1. just run `wordcounter` this fetches all comments
>
> 2. run `word counter -commentId=<commentID> -postId=<postID>` this filters the comments
>
>   sample output 
>
>    Starting Word Counter
      amet 24
      ducimus 31
      assumenda 35
      quibusdam 35
